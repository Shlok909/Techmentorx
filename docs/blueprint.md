# **App Name**: MentorLink

## Core Features:

- Role Selection: Allow users to select their role (Donor or Student/Volunteer) on the landing page and store it in localStorage.
- Donor Dashboard: Provide a dashboard for donors to post donation requests.
- Donation Posting: Enable donors to post donation requests with item name and pickup location, storing them in local state.
- Volunteer Dashboard: Provide a dashboard for student volunteers to view available donation requests.
- Donation List Display: Display a list of donation requests with item name, location, and status.
- Role-Based Redirection: Redirect users to their respective dashboards based on the role stored in localStorage.

## Style Guidelines:

- Primary color: Use a calming blue (#64B5F6) to evoke trust and community.
- Background color: Light gray (#F0F2F5) for a clean and unobtrusive backdrop.
- Accent color: A soft green (#A5D6A7) to highlight key actions like posting a donation.
- Body and headline font: 'Inter' sans-serif font for a modern, clean, and readable experience.
- Use a simple, card-based layout to clearly display donation requests and role options.
- Employ clear and intuitive icons to represent roles and donation information.
